The system accurately detects standing or walking humans in well-lit environments. Performance may vary depending on:
    Lighting conditions
    Camera quality
    Human pose and distance

 > Project detects humans in images, videos, and live webcam feed
 > Developed using Python and OpenCV
 > Uses HOG (Histogram of Oriented Gradients) with a pre-trained SVM classifier
 > Draws bounding boxes around detected humans
 > Suitable for surveillance and security applications
 > Simple and lightweight implementation
 > Works in real-time on webcam input
 > Requires Python 3, OpenCV, and NumPy
 > Easy to install and run
 > Can be extended using deep learning models like YOLO or SSD
